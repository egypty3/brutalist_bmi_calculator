# Brutal BMI Calculator 🏋️‍♂️

A Neo-Brutalist BMI calculator built with Flutter. This app features a high-contrast, bold design inspired by modern brutalism (Neo-Brutalism), characterized by thick borders, hard shadows, and vibrant colors.

## Features
- **Neo-Brutalist UI**: Premium custom widgets with solid shadows and high contrast.
- **Real-time BMI Calculation**: Instantly see your BMI as you adjust your height and weight.
- **Gender & Age Support**: Personalized calculations for fat percentage and ideal weight.
- **BMI Classification**: A clear breakdown of BMI categories.
- **Attractive Splash Screen**: Animated entry with `flutter_animate`.

## Tech Stack
- **Flutter**: UI Framework.
- **Google Fonts (Lexend)**: For bold, geometric typography.
- **Flutter Animate**: For smooth, high-quality transitions.

## Project Structure
- `lib/main.dart`: App theme and entry.
- `lib/screens/splash_screen.dart`: Animated entry screen.
- `lib/screens/calculator_screen.dart`: Main calculation interface.
- `lib/widgets/brutalist_widgets.dart`: Custom design components (Borders, Shadows).
- `lib/utils/bmi_logic.dart`: Scientific calculation formulas.

## Getting Started
1. Run `flutter pub get` to install dependencies.
2. Run `flutter run` to start the app.
