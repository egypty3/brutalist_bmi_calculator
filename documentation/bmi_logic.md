### 📐 1️⃣ الدالة `calculateBMI` - حساب مؤشر كتلة الجسم

**الوصف:**
تقوم بحساب مؤشر كتلة الجسم (BMI) باستخدام المعادلة الطبية المعتمدة عالمياً.

**الصيغة الرياضية:**
```
     الوزن بالكيلوجرام
BMI = -------------------
       (الارتفاع بالمتر)²
```

**الكود المصدري:**
<pre dir="ltr" style="text-align: left;"><code class="language-dart">/// يحسب مؤشر كتلة الجسم (BMI) باستخدام المعادلة الطبية المعتمدة عالمياً
double calculateBMI(double heightCm, double weightKg) {
  // ✅ التحقق من صحة المدخلات (يجب أن يكونا أكبر من الصفر)
  if (heightCm &lt;= 0 || weightKg &lt;= 0) return 0;
  
  // 🔄 تحويل الارتفاع من سم إلى متر
  double heightM = heightCm / 100;
  
  // 🧮 تطبيق المعادلة الرياضية
  return weightKg / (heightM * heightM);
}
</code></pre>

**أمثلة عملية:**

| الطول | الوزن | العملية الحسابية | النتيجة (BMI) |
|------|-------|------------------|---------------|
| 175 سم | 70 كجم | 70 ÷ (1.75 × 1.75) | **22.86** ✅ |
| 160 سم | 65 كجم | 65 ÷ (1.6 × 1.6) | **25.39** ⚠️ |

### 🎯 2️⃣ الدالة `getIdealWeightRange` - نطاق الوزن المثالي

**الوصف:**
تحسب **أدنى وأعلى** أوزان تعتبر "صحية" لشخص معين بناءً على طوله.

**المدخلات:**
| المدخل | الوحدة |
|--------|-------|
| الطول (heightCm) | سنتيمتر |

**المدخلات المطلوبة للحساب:**
| الحد الأدنى | الحد الأقصى |
|------------|-------------|
| BMI = 18.5 | BMI = 25.0 |

**الكود المصدري:**
<pre dir="ltr" style="text-align: left;"><code class="language-dart">/// يحسب وزن البداية والنهاية للنطاق الصحي (BMI بين 18.5 - 25)
static Map&lt;String, double&gt; getIdealWeightRange(double heightCm) {
  // 🔄 تحويل الارتفاع إلى متر
  double heightM = heightCm / 100;
  
  // 📏 الحد الأدنى للوزن الصحي (BMI = 18.5)
  double minWeight = 18.5 * (heightM * heightM);
  
  // 📏 الحد الأقصى للوزن الصحي (BMI = 25.0)
  double maxWeight = 25.0 * (heightM * heightM);
  
  // 📦 إرجاع البيانات في خريطة (Map)
  return {
    'min': minWeight,
    'max': maxWeight,
  };
}
</code></pre>

**أمثلة عملية:**

| الطول | الوزن المثالي (الحد الأدنى) | الوزن المثالي (الحد الأقصى) |
|------|---------------------------|---------------------------|
| 170 سم | **55.3 كجم** | **72.25 كجم** |
| 160 سم | **47.0 كجم** | **64.0 كجم** |
| 180 سم | **62.0 كجم** | **81.0 كجم** |

**التطبيق العملي:**
*"إذا كنت تطول 170 سم، فإن وزنك المثالي يتراوح بين **55.3 - 72.3** كجم!"

### 🩸 3️⃣ الدالة `estimateFatPercentage` - نسبة دهون الجسم

**الوصف:**
تقدر **نسبة الدهون في الجسم** باستخدام معادلة طبية معتمدة.

**الصيغة الطبية (معادلة Deurenberg):**
```
نسبة الدهون = (1.20 × BMI) + (0.23 × العمر) - (10.8 × الجنس) - 5.4
```

**جدول قيمة الجنس في المعادلة:**
| الجنس | القيمة المستخدمة |
|--------|-----------------|
| ذكر (Male) | **1** ✅ |
| أنثى (Female) | **0** ✨ |

**الكود المصدري:**
<pre dir="ltr" style="text-align: left;"><code class="language-dart">/// تقدر نسبة الدهون في الجسم باستخدام معادلة طبية معتمدة
static double estimateFatPercentage(double bmi, int age, bool isMale) {
  // ✅ التحقق من صحة BMI (يجب أن يكون موجباً)
  if (bmi &lt;= 0) return 0;
  
  // 🔄 تحويل الجنس إلى قيمة رقمية (1 للذكر، 0 للإناث)
  int genderVal = isMale ? 1 : 0;
  
  // 🧮 تطبيق المعادلة الرياضية
  return (1.20 * bmi) + (0.23 * age) - (10.8 * genderVal) - 5.4;
}
</code></pre>

**مثال عملي مفصل:**
```dart
// حساب نسبة الدهون لشخص ذكر عمره 30 عاماً وبمؤشر BMI = 25
double fatPercent = estimateFatPercentage(25, 30, true); // isMale = true

// حل المعادلة خطوة بخطوة:
// = (1.20 × 25) + (0.23 × 30) - (10.8 × 1) - 5.4
// = 30 + 6.9 - 10.8 - 5.4
// = **20.7%** 🎯
```

**تفسير النتيجة:**
*"نسبة الدهون 20.7% تعتبر نسبة صحية للأشخاص الذكور!"

### 🏷️ 4️⃣ الدالة `getClassification` - تصنيف حالة الجسم الصحية

**الوصف:**
يصنف نتيجة الـ BMI إلى **فئات صحية متعددة** مع توضيح كل فئة بلون مميز.

**جدول الفئات الطبية المعتمدة:**

| الفئة | النطاق (BMI) | اللون في التطبيق |
|------|-------------|------------------|
| **🔴 تحت الوزن - شديد** | < 16.0 | أحمر (#FF5252) |
| **🟠 تحت الوزن** | 16.0 - < 17.0 | برتقالي فاتح (#FF7043) |
| **🟠 تحت الوزن - خفيف** | 17.0 - < 18.5 | برتقالي داكن (#E65100) |
| **🟢 الوزن المثالي** | 18.5 - < 25.0 | أخضر (#4CAF50) ✅ |
| **🟡 فوق الوزن - خفيف** | 25.0 - < 30.0 | ذهبي (#F57F17) ⚠️ |
| **🧡 فوق الوزن - متوسط** | 30.0 - < 35.0 | برتقالي فاتح (#FF8A65) ⚠️ |
| **🟠 فوق الوزن** | 35.0 - < 40.0 | برتقالي داكن (#F4511E) ⚠️ |
| **🔴 السمنة - شديد** | ≥ 40.0 | أحمر داكن (#D32F2F) 🚨 |

**الكود المصدري:**
<pre dir="ltr" style="text-align: left;"><code class="language-dart">/// يصنف نتيجة الـ BMI إلى فئات صحية معروفة طبياً مع الألوان
static BMIResult getClassification(double bmi) {
  // 🔴 تحت الوزن - من الأكثر شدة إلى الأقل
  
  if (bmi &lt; 16) 
    return BMIResult('cat_vsu', const Color(0xFFFF5252)); // تحت الوزن - شديد
  
  if (bmi &lt; 17) 
    return BMIResult('cat_su', const Color(0xFFFF7043));  // تحت الوزن
  
  if (bmi &lt; 18.5) 
    return BMIResult('cat_u', const Color(0xFFE65100));  // تحت الوزن - خفيف
  
  // 🟢 الوزن المثالي
  if (bmi &lt; 25) 
    return BMIResult('cat_n', const Color(0xFF4CAF50));  // الوزن المثالي
  
  // 🟡🧡🟠 فوق الوزن - من خفيف إلى اعتدال
  if (bmi &lt; 30) 
    return BMIResult('cat_o', const Color(0xFFF57F17));  // فوق الوزن - خفيف
  
  if (bmi &lt; 35) 
    return BMIResult('cat_o1', const Color(0xFFFF8A65)); // فوق الوزن - متوسط
  
  if (bmi &lt; 40) 
    return BMIResult('cat_o2', const Color(0xFFF4511E)); // فوق وزن
  
  // 🔴 السمنة - شديد
  return BMIResult('cat_o3', const Color(0xFFD32F2F));  // سمنة - شديد
}
</code></pre>

**كيف تعمل أمثلة التطبيق:**
```dart
// ما هو تصنيف شخص بـ BMI = 28.5؟

if (bmi < 16) → ❌ لا ينطبق
if (bmi < 17) → ❌ لا ينطبق  
// ... يستمر في التحقق من الأدنى للأعلى
if (bmi < 25) → ❌ لا ينطبق (لأن 28.5 ≥ 25)
// ... يستمر في التحقق
if (bmi < 30) → ✅ **ينطبق!**

النتيجة:
- المفاتيح (Key): `'cat_o'`
- اللون: 🟡 `0xFFF57F17` (ذهبي)
```

## 📦 الكونة `BMIResult` - كونة بيانات (DTO)

**الوصف:**
كونة بسيط يربط بين مفتاح الترجمة واللون المرتبط بكل حالة صحية.

**الكود:**
<pre dir="ltr"><code class="language-dart">/// كونة بيانات يربط بين مفتاح الترجمة واللون المرتبط بكل حالة صحية
class BMIResult {
  final String key;   // مفتاح الترجمة (مثل 'cat_n'، 'cat_o')
  final Color color;  // اللون المرتبط بالحالة الصحية
  
  BMIResult(this.key, this.color);
}</code></pre>

**الاستخدام في الواجهة:**
<pre dir="ltr"><code class="language-dart">// عند عرض النتائج على الشاشة:
String categoryText = AppLocalization.getTranslation(bmiResult.key);
Color statusColor = bmiResult.color;</code></pre>

## 📚 جدول مرجعي سريع (Cheat Sheet)

| الدالة | متى تستخدمها؟ | العائد |
|-------|-------------|--------|
| `calculateBMI(height, weight)` | عند وجود الطول والوزن، تحتاج الـ BMI | double (0 إذا كانت القيم غير صحيحة) |
| `getIdealWeightRange(height)` | عند بدء المستخدم أو رغبة المستخدم في معرفة الأوزان المستهدفة | Map يحتوي على `'min'` و `'max'` |
| `estimateFatPercentage(bmi, age, isMale)` | عند توفر الـ BMI والعمر والجنس في عرض النتائج | double (النسبة المئوية 0-100) |
| `getClassification(bmi)` | عند الحاجة لعرض حالة الصحة مع اللون | BMIResult (مفتاح + لون) |
