## Full description
<en-GB>
Brutalist BMI Calculator is a bold, high-contrast BMI app built with Flutter.

Features:
- Instant BMI calculation as you adjust your height and weight
- Gender and age support for a more personalized result
- Estimated body fat percentage and ideal weight
- Clear BMI classification reference table
- Health-risk details for each BMI category
- Multi-language support: English, Arabic, French, and German
- Saved language preference so the app remembers your last choice
- Beginner-friendly tutorial screens and a localized splash screen

The app uses a clean Neo-Brutalist visual style with thick borders, strong shadows, and colorful accents to make the interface easy to read and visually distinctive.

This app is intended for educational and general wellness reference purposes.
</en-GB>
